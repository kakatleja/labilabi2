Задание 1.3. Имена

Алгоритм:
Создаётся сущность «Имя» с параметрами фамилия, личное имя и отчество. При преобразовании в строку выводятся только заданные параметры, отсутствующие параметры не учитываются.

Примеры вывода:

Клеопатра
Пушкин Александр Сергеевич

Задание 1.4. Время (секунды с начала суток)

Алгоритм:
Время задаётся количеством секунд с начала суток. Если количество секунд превышает 24 часа, используется остаток от деления на 86400. Полученное время выводится в формате ЧЧ:ММ:СС.

Примеры вывода:

0:00:10
3:20:00

Задание 2. Сотрудники и отделы

Алгоритм:
Создаются сущности «Сотрудник» и «Отдел». Сотрудник хранит ссылку на отдел, а отдел — на своего начальника. Формат вывода зависит от того, является ли сотрудник начальником отдела.

Примеры вывода:

Петров работает в отделе IT, начальник которого Козлов
Козлов начальник отдела IT

Задание 3. Сотрудники и отделы (список сотрудников)

Алгоритм:
Отдел хранит список всех сотрудников. При создании сотрудника он автоматически добавляется в список сотрудников своего отдела, что позволяет получить всех сотрудников по ссылке на любого из них.

Примеры вывода:

Петров
Козлов

Задание 4. Создание времени разными способами

Алгоритм:
Время создаётся либо по количеству секунд с начала суток, либо по заданным часам, минутам и секундам. Присваивание выполняется один раз за счёт использования делегирования конструкторов.

Примеры вывода:

3:20:00
2:03:05

Задание 5. Сколько сейчас времени

Алгоритм:
По количеству секунд с начала суток определяется текущий час, количество минут с начала часа и количество секунд с начала текущей минуты.

Примеры вывода:

Часы: 9
Минуты: 15
